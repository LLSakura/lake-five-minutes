# 湖边五分钟

一个治愈系像素风钓鱼网页小游戏。玩家在五分钟内点击抛竿、拖拽控制收线力度、收集不同季节的鱼、卖鱼获得金币，并升级鱼竿、鱼线、鱼饵和幸运值。

## 玩法

- 点击“抛竿”等待鱼上钩。
- 鱼上钩后，在画面中拖拽鼠标控制力道。
- 张力过高会断线，力度过低鱼会逃走。
- 把鱼卖掉获得金币，用金币升级装备。
- 每局会随机出现三个任务。
- 当前有 24 种鱼，不同季节会出现不同鱼池和季节特殊鱼。
- 图鉴支持按季节和稀有度筛选，已发现鱼会显示描述和基础信息。
- 本次浏览会话会保存图鉴和升级，刷新网页不会丢失；关闭后重新打开会重新开始。

## 本地运行

直接打开 `index.html` 即可游玩。

如果想用本地服务器运行：

```powershell
node server.js
```

然后打开 `http://127.0.0.1:4173/`。
